# ⛏️ Firefox Ore UI Theme

A custom `userChrome.css` theme for Firefox, meticulously designed to replicate the **Ore UI** design system from Minecraft Bedrock Edition.

---

## ✨ Features
* **Minecraft Visuals:** Authentic stone-textured backgrounds and pixel-perfect borders.
* **Ore UI States:** * 🟩 **Primary State:** Active tabs feature the signature Minecraft green button style.
  * 🟥 **Destructive State:** Close buttons turn red on hover.
* **Dynamic Animations:** Smooth "bounce" effect on tab hover and tactile scale-down effect on button clicks.
* **Pixel Typography:** Full support for the "Minecraftia" font across the entire UI.
* **Zero Border Radius:** Completely squared-off address bar and search fields for that classic blocky feel.

## 🚀 Installation

1. **Enable Custom Stylesheets:**
   * Open `about:config` in Firefox.
   * Search for `toolkit.legacyUserProfileCustomizations.stylesheets` and set it to **true**.
2. **Locate Profile Folder:**
   * Go to `about:support`.
   * Find **Profile Folder** and click **Open Folder**.
3. **Apply the Theme:**
   * Create a folder named `chrome` if it doesn't exist.
   * Place the `userChrome.css` file from this repository inside the `chrome` folder.
4. **Font (Required):**
   * Install the **Minecraftia** font on your system for the best experience.
5. **Restart Firefox.**
